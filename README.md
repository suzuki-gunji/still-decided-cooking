# 🍳アプリ名 
still-decided-cooking

## 💬概要　
- その日に作る料理が決まらなかった場合、ランダムに料理を表示して料理を決めることができる。
- その際麺類やご飯類などの細かなジャンルを決めて、料理の種類を選択することができる。
